TP : Création d'une page personnelle

Objectif de ce TP est de créer et de mettre en ligne un site internet personnel.

Ce site doit comprendre :

  • Une page d'accueil index.html contenant :
    • Une petite présentation académique (classe, spécialités,idées d'orientation...)
    • Une petite présentation personnelle (centres d'intérêts, sports pratiqués, soyez drôles)
    • Une image (photo de vous ou autre)
    • Une liste de navigation permettant d'accéder aux pages projets.html et rapport.html
  • Une page projets.html contenant :
    • Une description de quelques programmes que vous avez écrit en cours de NSI
    • Un accès à ces programmes
    • Une liste de navigation permettant d'accéder aux pages index.html et rapport.html
  • Une page rapport.html contenant un court rapport expliquant ce que vous avez appris en HTML et en CSS pour créer votre site web.
  • Un fichier CSS style.css permettant de personnaliser le style de la page.

Pour réaliser ce travail, vous devez apprendre à utiliser deux langages que sont HTML et CSS. Pour cela, vous allez consulter les pages bases en HTML et bases en CSS de la fondation Mozilla. Vous utiliserez l'éditeur Brackets (ou similaires) afin de coder en HTML et CSS.

À partir des menus déroulants sur la gauche de la page de Mozilla, vous êtes libres de découvrir le HTML et le CSS aussi loin que vous le souhaitez mais n'étudiez pas le Javascript !!

Un exemple -loin d'être parfait- de ce qui est demandé est disponible ici. Vous pouvez étudier le code de cette page en cliquant sur "Afficher, Options pour les développeurs, CodeSource".

Procédure de mise en ligne :
  • S'inscrire sur GitHub avec votre prénom et votre nom
  • Se connecter sur votre github
  • Créer un nouveau dossier appelé 'nom_de_compte.github.io'
  • Y déposer tous les fichiers de votre arborescence web
  • Votre site est disponible à l'adresse 'nom_de_compte.github.io'
